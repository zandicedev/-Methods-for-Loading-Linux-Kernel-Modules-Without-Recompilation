# Methods for loading linux kernel modules without recompilation
This project was carried out as part of my end-of-studies internship at the University of Bordeaux, following and extending my final-year project.
Made by zandicedev.

## Overview
This work focuses on the analysis of Linux kernel module loading and validation mechanisms, with particular attention to compatibility checks and module verification processes.

The study covers the following topics:
- Checks performed during kernel module loading
- Kernel module loading and validation diagrams
- Analysis of a tool for modifying module vermagic and CRC values
